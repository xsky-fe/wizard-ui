# DatePicker
<em><strong>(wizard-ui自定义组件)</strong></em>
<p>2023-6-19 [peishan](https://github.xsky.com/peishan)</p>
展示月历，允许用户选择日期和具体的时间。基于 [rc-calendar](https://github.com/react-component/calendar) 和[rc-time-picker](https://github.com/react-component/time-picker)开发。

## 使用场景
1. 选择单个日期；
2. 默认支持选择日期和具体到秒级别的时间，可以定制只选择日期；
3. 支持中文、英文。
