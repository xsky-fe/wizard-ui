# Pagination
<em><strong>(react-bootstrap扩展组件)</strong></em>
<p>2023-6-15 [chensheng](https://github.xsky.com/chen-sheng)</p>
采用分页的形式分隔长列表，每次只加载一个页面。

## 使用场景
- 当加载/渲染所有数据将花费很多时间时；
- 可切换页码浏览数据。
## 基本用法
### 交互
- `<<` 首页
- `>>` 最后一页
- `<` 上一页
- `>` 下一页
