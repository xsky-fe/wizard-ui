# RangePicker
<em><strong>(wizard-ui自定义组件)</strong></em>
<p>2023-6-19 [peishan](https://github.xsky.com/peishan)</p>

选定开始时间和结束时间输出时间区间。基于 [rc-calendar](https://github.com/react-component/calendar) 和[rc-time-picker](https://github.com/react-component/time-picker)开发。

## 基本用法
1. 开始时间小于结束时间。
2. 支持选择日期同时支持选择时间，精确到秒级别。
3. 支持中文、英文。默认可以根据浏览器的本地存储变量LOCALE决定切换。
4. 支持区间（3 小时内、1 天内、1 周内、1 月内、3 月内）选择，选择区间会立即关闭选择面板。
