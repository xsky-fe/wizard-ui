# TimePicker
用户输入一个时间，可以点击标准输入框，弹出时间面板进行选择。基于 [rc-time-picker](https://github.com/react-component/time-picker) 开发。
## 基本用法
### 默认展示
<Canvas of={TimePickerStories.Basic}/>
## 便利用法
### 分钟固定为 0 的小时选择
选择时间区间时，设置 hourStart 选择开始时间。设置 hourEnd 选择结束时间.
<Canvas of={TimePickerStories.SelectHour}/>
### 禁止选择小时
<Canvas of={TimePickerStories.DisabledHour}/>
