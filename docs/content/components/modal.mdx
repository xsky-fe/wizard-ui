对话弹出框。

## 何时使用
在处理某个事务，不希望跳转到其他页面导致当前工作被打断。可以使用`对话框`在当前页面打开一个浮层，承载对应的操作。

## 基本用法
### 默认展示
```jsx
() => {
  const [ show, setShow ] = React.useState(false);
  const handleClick = () => setShow(true);
  const handleHide = () => setShow(false);
  return (
    <div>
      <Button variant="primary" onClick={handleClick}>打开对话框</Button>
      <Modal
        title="基本对话框"
        show={show}
        onHide={handleHide}
        onOk={handleHide}
      >
        <p>内容 1</p>
        <p>内容 2</p>
      </Modal>
    </div>
  )
}
```

() => {

### 异步提交关闭
模拟异步提交，例如提交表单。
```jsx
() => {
  const [ show, setShow ] = React.useState(false);
  const [ loading, setLoader ] = React.useState(false);
  const handleClick = () => setShow(true);
  const handleHide = () => setShow(false);
  const handleOk = () => {
    setLoader(true);
    setTimeout(() => {
      setLoader(false);
      setShow(false);
    }, 3000);
  }
  return (
    <div>
      <Button variant="primary" onClick={handleClick}>异步提交</Button>
      <Modal
        title="异步提交"
        show={show}
        onHide={handleHide}
        onOk={handleOk}
        loading={loading}
      >
        <p>点击“确认”对话框会在 3 秒后关闭。</p>
      </Modal>
    </div>
  )
}
```

### 改变样式
可以改变提交按钮的文本和颜色。比如“删除”场景
```jsx
() => {
  const [ show, setShow ] = React.useState(false);
  const handleClick = () => setShow(true);
  const handleHide = () => setShow(false);
  return (
    <div>
      <Button variant="danger" onClick={handleClick}>删除操作</Button>
      <Modal
        title="删除 xxx"
        show={show}
        onHide={handleHide}
        onOk={handleHide}
        okStyle="danger"
        confirmText="删除"
      >
        <p>内容 1</p>
        <p>内容 2</p>
      </Modal>
    </div>
  )
}
```
### 改变宽度
```jsx
() => {
  const [ show, setShow ] = React.useState(false);
  const handleClick = () => setShow(true);
  const handleHide = () => setShow(false);
  const SIZES = [['sm', '小'], [undefined, '中'], ['lg', '大']];
  const [ size, setSize ] = React.useState(undefined);
  return (
    <div>
      <Card>
        {SIZES.map(([val, text], idx) => (
          <Radio
            key={idx}
            inline
            checked={val === size}
            value={val}
            onChange={() => {
              if (val !== size) {
                setSize(val);
              }
            }}
          >
            {`${text}(${val})`}
          </Radio>
        ))}
      </Card>
      <Button variant="primary" onClick={handleClick}>打开对话框</Button>
      <Modal
        title="基本对话框"
        show={show}
        onHide={handleHide}
        onOk={handleHide}
        bsSize={size}
      >
        <p>内容 1</p>
        <p>内容 2</p>
      </Modal>
    </div>
  )
}
```
