作为 Input 一部分的下拉框

## 使用方式
通常和 Input 一起使用，在 Input 的前面或者后部，用来增强 Input 功能

## 基本用法

```jsx
() => {
  const [value, updateValue] = React.useState('MB');
  return (<div style={{ width: '300px' }}>
    <InputGroup>
      <FormControl />
      <InputDropdown
        pullRight
        defaultValue="GB"
        onChange={updateValue}
        value={value}
        options={[
          { title: 'MB', value: 'MB' },
          { title: 'GB', value: 'GB' },
          { title: 'TB', value: 'TB' },
        ]} />
    </InputGroup>
  </div>);
}
```
