将 well 用在元素上，能显示嵌入（inset）的效果，与面板（panel）有些类似，不过 panel 功能更丰富。

## 何时使用

- 增强文案显示
- 提示文案
- 与上下文做区分

### 常规

```jsx
<div>
  <Well>Look I'm in a well!</Well>
</div>
```

### 大间距

```jsx
<div>
  <Well bsSize="large">Look I'm in large well!</Well>
  <Well bsSize="lg">Look I'm in lg well!</Well>
</div>
```

### 小间距

```jsx
<div>
  <Well bsSize="sm">Look I'm in sm well!</Well>
  <Well bsSize="small">Look I'm in small well!</Well>
</div>
```

## API

| 参数    | 类型                      | 默认值 | 说明     |
| ------- | ------------------------- | ------ | -------- |
| bsSize  | "lg"/"large"/"sm"/"small" |        | 尺寸大小 |
| bsClass | string                    | 'well' | 样式     |
