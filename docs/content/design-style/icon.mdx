由 [wizard icons](https://github.com/xsky-fe/icons) 通过 react svg 组件方式提供。

## 使用场景
直观展示某个资源或者某种状态。

## 代码演示

### 常规
```jsx
<Icon type="os-search-role"/>
```

### 颜色
```jsx
<Icon type="os-search-role" color="primary" />
```
### 默认继承外部字体大小
默认继承父级字体大小和颜色。
```jsx
<div style={{ fontSize: '40px', color: '#6b58c4' }}>
  <Icon type="os-search-role" />
</div>
```
